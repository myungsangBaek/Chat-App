# WebRTC & WebSockets App

### HTTP Protocol

서버가 client를 기억 못하는 것을 **'stateless'** 라고 한다.

일반적인 http통신은 **'stateless'** 이다.

서버는 오직 request를 받을때만 response를 준다.

### WebSocket

WebSocket은 악수와 같다. 한 번 연결되면 손 잡고있는 것처럼 서로 커뮤니케이션이 가능하다.

연결되어있기에 서버는 유저가 누군지 알 수 있다.

서버는 request를 기다릴 필요가 없다.

브라우저에는 내장된 webSocket API가 있다.

**bi-direction**
